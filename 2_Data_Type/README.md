# Data Types in Programming

## Introduction
Data types define the type of data that a variable can store. They determine the kind of values that can be stored and what operations can be performed on them.

---

## 1. Integer (int)
An integer data type stores whole numbers (positive, negative, or zero) without decimal points.

**Examples:**
- 10  
- -5  
- 0  

**Use Case:** Counting items, indexing, loop counters.

---

## 2. Floating-Point (float / double)
A floating-point data type stores numbers with decimal points.

**Examples:**
- 3.14  
- -0.99  
- 10.0  

**Use Case:** Measurements, percentages, scientific calculations.

---

## 3. Character (char)
A character data type stores a single character such as a letter, digit, or symbol.

**Examples:**
- 'A'  
- '7'  
- '#'  

**Use Case:** Storing individual letters or symbols.

---

## 4. String (string)
A string data type stores a sequence of characters (text).

**Examples:**
- "Hello"  
- "12345"  
- "Data"  

**Use Case:** Names, messages, text processing.

---

## 5. Boolean (bool)
A boolean data type stores only two values: true or false.

**Examples:**
- true  
- false  

**Use Case:** Conditional statements, logical operations.

---

## 6. Array
An array is a collection of elements of the same data type stored in contiguous memory locations.

**Example:**
- [1, 2, 3, 4, 5]

**Use Case:** Storing multiple values of the same type.

---

## 7. Object
An object is a complex data type that stores data in key-value pairs.

**Example (JSON format):**
```json
{
  "name": "John",
  "age": 25
}
```

**Use Case:** Representing real-world entities.

---
